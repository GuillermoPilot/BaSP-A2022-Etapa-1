Trackgenix

By GigaTech Software Solutions SA

"The company GigaTech Software Solutions SA provides software development services to other companies, which have too many employees. Because of this "Trackgenix" was created “Trackgenix” is a web page that facilitates the registration of hours worked by each employee in each project and provides information to the companies so they can make better decisions.
